# Documentation

Bash script for web deployment in php

## Clone Repo

```bash
$ git clone https://github.com/KiranPranay/web_scripts scripts
$ cd scripts
```

## Setup php

```bash
$ bash setup.sh
```

## Host Multiple Websites

```bash
$ bash add_site.sh
```
